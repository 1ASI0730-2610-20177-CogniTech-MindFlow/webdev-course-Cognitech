# Fundamentos de Desarrollo Web para Principiantes — Cognitech

Repositorio de código fuente del curso introductorio de Desarrollo Web (HTML y CSS) dirigido a estudiantes de secundaria (de 12 a 17 años).  
Sin instalaciones. Solo un navegador web.

## Estructura del repositorio

```text
webdev-course-[nombre-del-equipo]/
├── course-plan.md              # Plan completo del curso (artefacto principal)
└── lessons/
    ├── lesson-01/              # Introducción al desarrollo web: ¿qué es un sitio web?
    │   ├── theory.md           # Contenido teórico + enlace a práctica online
    │   ├── starter/            # Código inicial para el estudiante
    │   └── completed/          # Código completo de referencia
    ├── lesson-02/              # HTML básico: etiquetas, elementos y atributos
    ├── lesson-03/              # Elementos HTML comunes: títulos, párrafos, listas, imágenes y enlaces
    ├── lesson-04/              # Introducción a CSS: colores, fuentes, selectores y bordes
    └── lesson-05/              # Proyecto final: crear una página de perfil + consejos y errores comunes
```
