## Contenido teórico

### ¿Qué es HTML?

HTML es el lenguaje que usamos para crear la estructura de una página web.

Podemos imaginar una página web como una casa. HTML sería la estructura principal: las paredes, puertas y ventanas. Todavía no tiene decoración, pero permite ordenar todo lo que aparecerá en la página.

Con HTML podemos colocar títulos, textos, imágenes, enlaces, listas y otros elementos.

---

### ¿Qué es una etiqueta HTML?

Una etiqueta HTML es una palabra especial que le indica al navegador qué tipo de contenido debe mostrar.

Por ejemplo, la etiqueta `<h1>` sirve para mostrar un título principal y la etiqueta `<p>` sirve para mostrar un párrafo.

La mayoría de etiquetas tienen una apertura y un cierre.

<h1>Mi primera página web</h1>
