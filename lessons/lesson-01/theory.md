## Contenido teórico

### ¿Qué es un sitio web?

Un sitio web es un conjunto de páginas y archivos que un navegador web interpreta para mostrar información al usuario.

Cuando visitamos páginas como YouTube, Wikipedia o cualquier otro sitio de Internet, realmente estamos visualizando contenido generado a partir de archivos que contienen texto, imágenes, enlaces y estilos visuales.

Los sitios web permiten compartir información, ofrecer servicios y crear experiencias interactivas para millones de usuarios alrededor del mundo.

---

### ¿Qué son HTML y CSS?

Para construir una página web utilizamos diferentes tecnologías. Las dos más importantes para comenzar son HTML y CSS.

**HTML (HyperText Markup Language)** se encarga de la estructura y el contenido de una página. Con HTML podemos agregar títulos, párrafos, botones, imágenes y otros elementos.

**CSS (Cascading Style Sheets)** se encarga de la apariencia visual. Con CSS podemos cambiar colores, tamaños, fuentes, alineaciones y otros aspectos relacionados con el diseño.

Una forma sencilla de entender la diferencia es imaginar una casa:

- HTML representa la estructura de la casa.
- CSS representa la pintura, decoración y estilo visual.

Ambas tecnologías trabajan juntas para crear sitios web atractivos y funcionales.

---

## ¿Cómo luce el código?

### HTML

```html
<h1>💻 Desarrollo Web</h1>

<p>
  Aprende a crear páginas web desde cero
  utilizando HTML y CSS.
</p>

<button>Comenzar</button>
```

### CSS

```css
body{
  font-family: Arial, sans-serif;
  text-align: center;
  background-color: #eef5ff;
  padding-top: 100px;
}

h1{
  color: #1e40af;
}

button{
  background-color: #2563eb;
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 8px;
}
```

---

## ¿Qué hace este ejemplo?

En este ejemplo:

- La etiqueta `h1` crea un título principal.
- La etiqueta `p` crea un párrafo.
- La etiqueta `button` crea un botón.
- CSS modifica la apariencia visual de la página.
- CSS permite cambiar colores, fuentes y estilos.

---

## Resultado esperado

Al ejecutar el código se mostrará una página sencilla con:

- Un título principal.
- Un texto descriptivo.
- Un botón.
- Un fondo de color suave.
- Elementos centrados en pantalla.

## Actividad práctica

### Tu primera página web

1. Ingresa a CodePen.
2. Copia el código HTML en el panel HTML.
3. Copia el código CSS en el panel CSS.
4. Observa cómo cambia la apariencia de la página.
5. Modifica el texto y los colores para personalizar tu diseño.

---

## Conclusiones clave

- Un sitio web es un conjunto de archivos interpretados por un navegador.
- HTML define el contenido y la estructura.
- CSS define la apariencia visual.
- No es necesario instalar programas para comenzar a desarrollar páginas web.
- Con pocas líneas de código es posible crear una página funcional y atractiva.
